# Brand Guidelines

## Logo

The Mobile Gaming logo features the letters "MG" in a gradient from cyan to coral, set against a dark background.

### Logo Variations
- **Full Color** - Use on dark backgrounds
- **Light** - Use on light backgrounds (white text)
- **Icon Only** - For small sizes (32x32 and below)

### Clear Space
Maintain minimum clear space equal to the height of the "M" around the logo.

### Minimum Size
- Print: 0.5 inches / 12mm
- Digital: 32x32 pixels

## Iconography

Use emoji for game representation:

### UI Icons
Use simple, geometric icons:
- Stroke width: 2px
- Size: 24x24 default, 20x20 for dense UI
- Color: Current text color or accent

## Writing Style

### Voice
- **Friendly but professional** - Casual enough for gaming, polished enough for broad appeal
- **Clear and direct** - No jargon, no fluff
- **Honest** - Match the brand promise of "what you see is what you get"

### Taglines
- Primary: "Play the games the ads promised"
- Alternative: "Browser puzzle games that deliver"
- Feature: "No ads. No tracking. Just puzzles."

### Headline Style
- Use sentence case for most headlines
- Keep headlines under 10 words
- Lead with benefits, not features

### Body Copy
- Keep paragraphs short (2-3 sentences)
- Use bullet points for lists of 3+ items
- Avoid superlatives (best, amazing, incredible)

## Description

Mobile Gaming is a collection of browser-based puzzle games that deliver exactly what mobile game advertisements promise. Built as a Progressive Web App, it offers four satisfying puzzle types without ads, tracking, or paywalls.

### Short Description (50 words)

Mobile Gaming offers four browser-based puzzle games - Water Sort, Parking Escape, Bus Jam, and Pull the Pin - that match the satisfying gameplay shown in mobile ads. No bait-and-switch, no ads, no tracking. Just pure puzzle satisfaction. Works offline, plays on any device.

### Long Description (150 words)

After years of watching mobile game advertisements show satisfying puzzle gameplay that never matched the actual games, Mobile Gaming was created to deliver on those promises.

The collection features four distinct puzzle types:
- **Water Sort** - Pour colored liquids between tubes to sort them into single-color containers
- **Parking Escape** - Classic sliding block puzzles where you guide cars to the exit
- **Bus Jam** - Match color-coded passengers to their corresponding buses
- **Pull the Pin** - Remove pins in the correct order to guide balls to their goal

Built as a Progressive Web App, Mobile Gaming works on any device with a browser. Install it for offline play, enjoy ad-free gameplay, and track your progress across all games. With 100+ handcrafted levels per game and procedurally generated infinite mode, there's always a new puzzle to solve.

No ads. No tracking. No paywalls. Just puzzles.
